# Environment Variables

Copy this file to `.env` and fill in your values before starting the stack.

```sh title=".env"
# ── Authentication ─────────────────────────────────────────────────────────────
# Generate with: openssl rand -hex 32
JWT_SECRET=

# ── AudioBookShelf ─────────────────────────────────────────────────────────────

# API token for server-to-server ABS calls (optional).
# Required for automatic multi-part audiobook merge on library clean.
# ABS → Settings → Users → your user → API Token.
ABS_API_KEY=

# ── Prowlarr ───────────────────────────────────────────────────────────────────
PROWLARR_API_KEY=

# ── qBittorrent ────────────────────────────────────────────────────────────────
QBIT_USERNAME=
QBIT_PASSWORD=

# ── Notifications (optional) ───────────────────────────────────────────────────
DISCORD_WEBHOOK_URL=
```
